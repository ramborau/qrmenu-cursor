# Model Context Protocol (MCP) References

This document contains all MCP (Model Context Protocol) references used across the QR Menu project. These MCPs provide AI agents with comprehensive context about the technologies and frameworks used.

## Available MCPs

1. **[Next.js](https://context7.com/vercel/next.js/llms.txt)** - Next.js framework documentation and best practices
2. **[React](https://context7.com/reactjs/react.dev)** - React library documentation and patterns
3. **[ShadCN/UI](https://context7.com/shadcn-ui/ui/llms.txt)** - ShadCN/UI component library documentation
4. **[Better Auth](https://context7.com/better-auth/better-auth/llms.txt)** - Better Auth authentication library documentation
5. **[Prisma](https://context7.com/prisma/docs/llms.txt)** - Prisma ORM documentation and schema design

## Usage

These MCPs should be referenced when:

-   Building frontend components (Next.js, React, ShadCN/UI)
-   Implementing authentication flows (Better Auth)
-   Designing database schemas (Prisma)
-   Making architectural decisions involving these technologies

## Agent MCP Mapping

-   **Frontend Agent:** Next.js, React, ShadCN/UI
-   **Backend Agent:** Next.js, Better Auth
-   **Database Agent:** Prisma
-   **Architect Agent:** All MCPs (for comprehensive technical decisions)
